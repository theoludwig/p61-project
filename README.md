## Prérequis

- [Node.js](https://nodejs.org/) >= 20.0.0
- [npm](https://www.npmjs.com/) >= 10.0.0
- [Expo Go](https://expo.io/client)

## Installation

```sh
npm clean-install
```

## Usage

```sh
# Run the application
npm run start

# Lint and Test
npm run lint:prettier
npm run lint:eslint
npm run lint:typescript
npm run test
```
